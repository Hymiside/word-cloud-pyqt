Техническое задание

Цель:
Реализовать приложение генерируещее паттерн в .png
формате из переписки в Telegram.

Декомпозиция:
1. Написать алгоритм реализующий генерацию паттерна
2. Написать интерфейс
    2.1 Создать схемы БД и реализовать их
    2.2 Написать виджет входа в приложение
        2.2.1 Написать UI виджета
        2.2.2 Написать логику для UI
    2.3 Написать виджет регистрации в приложении
        2.3.1 Написать UI виджета
        2.3.2 Написать логику для UI
    2.4 Написать виджет главного экрана
        2.4.1 Написать UI виджета
        2.4.2 Написать логику для UI
    2.5 Написать виджет инструкции для пользователя
3. Реализовать алгоритм генерации внутри интерфейса приложения
4. Рефактор кода
5. Сдать рабочее приложение
